## 1. Миссия

Создать первый региональный инструмент мониторинга эмиссии метана из болот Западной Сибири, связывающий спутниковые наблюдения атмосферного CH₄ с наземными данными и типологической картой болот, и предоставляющий результат как открытый воспроизводимый инструмент с визуальным интерфейсом.

## 2. Доменная модель (онтология)

### 2.1. Сущности

- **Пиксель атмосферного CH₄** — единица спутникового наблюдения колонковой концентрации метана (XCH₄). Площадь покрытия ~50 км². Содержит интегральную информацию по всему столбу атмосферы, не только приземному слою.

- **Аномалия CH₄ (ΔCH₄)** — разность между наблюдённым XCH₄ и фоновым значением. Фон — характеристика региона без болотных источников. Аномалия — proxy для поверхностной эмиссии.

- **Болотный микроландшафт** — минимальная типологическая единица болотного покрова (гряда, мочажина олиготрофная, мочажина мезотрофная, мочажина эвтрофная, рям, мочажинное озерко, сплавина). Каждый тип имеет характерный диапазон эмиссии CH₄.

- **Карта болот с типологией** — пространственный слой, задающий для каждого пикселя подстилающей поверхности тип болотного микроландшафта или отсутствие болота. Разрешение карты на порядки выше разрешения атмосферных данных.

- **Наземная станция** — точка с измеренными потоками CH₄ (eddy covariance или камерный метод) и сопутствующими метеоданными. Служит для калибровки и валидации.

- **Предиктор** — пространственно-временная переменная, объясняющая вариацию ΔCH₄: температура почвы, доля инундированной площади, вегетационный индекс, тип болота, широта, день года.

- **Модель proxy-эмиссии** — статистическая связь между ΔCH₄ и набором предикторов, калиброванная по наземным данным. Не физическая модель переноса, а эмпирический инструмент.

### 2.2. Фундаментальные разделения

- **XCH₄ ≠ поверхностный поток.** Колонковая концентрация интегрирует весь столб атмосферы. Прямая конверсия в поток невозможна без модели переноса. Проект работает с аномалиями (enhancement), а не абсолютными концентрациями.

- **Карта болот ≠ карта эмиссии.** Карта задаёт WHERE и WHAT TYPE, но не HOW MUCH. Интенсивность эмиссии зависит от метеоусловий и сезона.

- **Камерные данные ≠ EC-данные.** Камеры — точечные, EC — интегральные (~1 км²). Масштабы не совпадают. При калибровке необходим учёт footprint-а EC-мачты и доли микроландшафтов в нём.

- **Сезонный сигнал ≠ детекция источника.** TROPOMI XCH₄ над болотами ЗС отражает преимущественно сезонную динамику (май–октябрь). Инструмент мониторирует сезонно-пространственную вариацию, а не детектирует аномальные выбросы.

### 2.3. Типизации

**Микроландшафты болот** (по модели Bc8 Glagolev/Sabrekov):
1. Пальсы (palsas)
2. Рямы (ryams) — сосново-кустарничково-сфагновые
3. Гряды (ridges)
4. Мочажины олиготрофные (oligotrophic hollows)
5. Мочажины мезотрофные (mesotrophic hollows)
6. Мочажины эвтрофные (eutrophic hollows)
7. Сплавины (peat mats)
8. Мочажинные озёрки (wetland ponds)

**Зоны ЗС** (с юга на север): лесостепь, подтайга, южная тайга, средняя тайга, северная тайга, лесотундра, тундра.

## 3. Ограничения (деонтика)

### 3.1. Только аномалии, не абсолютные концентрации

Инструмент вычисляет и показывает ΔCH₄, а не XCH₄. Фоновое значение вычитается. Без этого шага любые выводы об эмиссии из болот невалидны.

### 3.2. Карта болот первична

Без карты болот с типологией инструмент — просто визуализация TROPOMI. Карта болот задаёт маску анализа и субпиксельную декомпозицию. Она должна существовать до построения модели.

### 3.3. Калибровка только по опубликованным данным

Наземные данные для калибровки берутся из рецензированных публикаций (Dyukarev et al. 2024, Sabrekov et al. 2011/2013, Veretennikova & Dyukarev 2021) и открытых датасетов (Zenodo doi:10.5281/zenodo.4323024). Неопубликованные данные — только с формальным согласием PI.

### 3.4. Честность относительно ограничений

Документация и статья должны явно указывать:
- TROPOMI не видит приземный CH₄ напрямую
- Зима (ноябрь–апрель) не покрыта данными
- Одна калибровочная станция (Мухрино) не репрезентирует всю ЗС
- Разрешение TROPOMI (~7 км) не позволяет различить микроландшафты напрямую

### 3.5. Открытость и воспроизводимость

Весь код — открытый. GEE-скрипт — публикуемый. Данные — доступные или с указанием как получить. Результат воспроизводим любым исследователем с аккаунтом GEE.

### 3.6. Никаких заявлений о верификации национальных кадастров

Инструмент — исследовательский. Точность недостаточна для верификации отчётности по Парижскому соглашению. Формулировки типа «инструмент для проверки национальных инвентаризаций» запрещены.

## 4. Критерии качества (аксиология)

### 4.1. Карта болот
- Общая точность классификации ≥ 85% (по confusion matrix с полевой валидацией или VHR imagery)
- Минимум 5 типов микроландшафтов различимы
- Покрытие: 55–68°N, 60–85°E

### 4.2. Модель proxy-эмиссии
- R² ≥ 0.4 по leave-one-out cross-validation (учитывая что WetCH4 global даёт 0.51)
- RMSE в пределах опубликованных значений для бореальных болот
- Сезонная динамика (пик в июле–августе) воспроизводится

### 4.3. Инструмент (GEE App)
- Время загрузки карты < 10 секунд
- Слайдер времени работает плавно
- Экспорт данных в CSV/GeoTIFF функционирует

### 4.4. Приоритеты

```
Научная корректность > Воспроизводимость > Полнота покрытия > Удобство интерфейса > Красота визуализации
```

## 5. Принципы (праксеология)

### 5.1. Экономическая модель

Полностью бесплатный инструмент. GEE — бесплатен для исследований. TROPOMI — открытые данные Copernicus. Sentinel-2 — открытые данные. Наземные данные — из публикаций или open-access репозиториев.

### 5.2. Масштаб

- Территория: Западно-Сибирская низменность (55–68°N, 60–85°E)
- Время: 2019–present (определяется доступностью TROPOMI)
- Разрешение: месячные композиты
- Пользователи: 10–100 исследователей в год (реалистичная оценка для нишевого инструмента)

### 5.3. Эволюция

**Фаза 1 (MVP):** Карта болот + TROPOMI anomaly map + визуализация в GEE
**Фаза 2:** Модель proxy-эмиссии с RF + калибровка по Мухрино
**Фаза 3:** GEE App с интерактивным интерфейсом + data paper
**Фаза 4 (future):** Расширение калибровочной сети, добавление GOSAT/MethaneSAT

## 6. Специфика предметной области

### 6.1. Болота ЗС — крупнейший в мире ненарушенный болотный массив

Площадь болот ~1.3 млн км². Преобладают олиготрофные raised bogs с грядово-мочажинным комплексом. Зональность выражена резко: от лесостепных болот на юге (56°N) до тундровых на севере (68°N). Каждая зона имеет свой режим эмиссии CH₄.

### 6.2. Мухрино — ключевой, но единственный полноценный калибровочный сайт

60.89°N, 68.68°E. Олиготрофный raised bog средней тайги. EC с 2015 (CO₂), CH₄ с 2019 (LI-7700). Камерные данные с автоматической системой КАСМ8 (Dyukarev et al. 2024). Метеоданные 2010–2019 на Zenodo. Репрезентативен для одного типа болот одной зоны.

### 6.3. TROPOMI над ЗС — специфические ограничения

- Облачность: май–октябрь ~50% потерь, зимой ~90%
- Снежный покров: ноябрь–апрель — практически нет данных CH₄
- Высокие широты: качество ретривала XCH₄ ухудшается при низком солнце (SZA > 70°)
- Болота и water bodies: qa_value для пикселей над водой часто < 0.5

### 6.4. Субпиксельная гетерогенность — ключевая методологическая проблема

Один пиксель TROPOMI (~50 км²) содержит мозаику: гряды, мочажины, рямы, озёрки, лес, реки. Каждый элемент мозаики эмитирует CH₄ по-разному. Типологическая карта болот — единственный способ декомпозиции пиксельного сигнала.

## 7. Типовые запросы / сценарии

1. **Исследователь:** «Какова пространственная картина аномалий CH₄ над болотами ЗС в июле 2023?» → Карта ΔCH₄ с фильтром по дате.

2. **Исследователь:** «Как соотносятся аномалии CH₄ с типами болот?» → Boxplot или scatter ΔCH₄ vs доля микроландшафта.

3. **Рецензент статьи:** «Воспроизведите результаты» → Клонировать GEE-скрипт, запустить с теми же параметрами, получить те же карты.

4. **Модельер:** «Дайте мне данные для валидации моей модели» → Экспорт ΔCH₄ + предикторов в CSV/GeoTIFF для AOI.

5. **Студент:** «Покажите тренд эмиссии CH₄ из болот ЗС за 2019–2025» → Таймсерия среднего ΔCH₄ по региону с доверительным интервалом.
