## 1. Миссия

Создать региональный инструмент мониторинга эмиссии метана из болот Западной Сибири на основе данных TROPOMI, сопоставляющий спутниковые наблюдения XCH₄ с картой болот, температурой и вегетационными индексами, и позволяющий оценить уровень эмиссии с валидацией по наземным станциям.

---

## 2. Доменная модель (онтология)

### 2.1. Сущности

- **XCH₄** — column-averaged dry-air CH₄ mixing ratio (ppb). Основной измеряемый параметр TROPOMI. Центральный вход инструмента.

- **Enhancement (ΔCH₄)** — превышение XCH₄ над региональным фоном. Показывает вклад локальных поверхностных источников.

- **Болотный микроландшафт** — минимальная типологическая единица болотного покрова (8 типов по Bc8).

- **Карта болот с типологией** — пространственный слой. Роли: маска, вычисление фона, атрибуция по типам, субпиксельная декомпозиция.

- **Три ключевых показателя** для сопоставления с наземными данными:
  1. XCH₄ / ΔCH₄ из TROPOMI — уровень метана
  2. Температура воздуха — из ERA5 — управляющий фактор метаногенеза
  3. NDVI — из MODIS/Sentinel-2 — proxy продуктивности и субстрата

- **Наземная станция** — точка с измеренными потоками CH₄ (EC или камеры) + метео. Для валидации связи TROPOMI ↔ surface flux.

- **Относительный вклад** — enhancement CH₄ над болотами относительно лесов (ΔCH₄_болота − ΔCH₄_леса). Не требует конверсии в flux.

- **Абсолютный вклад** — оценка surface flux через сопоставление с наземной станцией. Требует transfer function.

### 2.2. Фундаментальные разделения

- **XCH₄ ≠ поверхностный поток.** Но XCH₄ над болотами > XCH₄ над лесами → относительный вклад определяется напрямую.

- **Мониторинг сначала, моделирование потом.** Сначала научиться собирать и агрегировать TROPOMI по болотам. Потом — сопоставлять с наземными. Потом — моделировать.

- **Относительный вклад не требует конверсии.** Абсолютный — требует, но это второй этап.

### 2.3. Типизации

**Микроландшафты** (Bc8): пальсы, рямы, гряды, мочажины олиго/мезо/эвтрофные, сплавины, озёрки.

**Зоны ЗС:** лесостепь → подтайга → южная тайга → средняя тайга → северная тайга → лесотундра → тундра.

## 3. Ограничения (деонтика)

### 3.1. TROPOMI — центральный инструмент мониторинга

TROPOMI XCH₄ — основной источник данных. Пайплайн:

```
Этап 1: TROPOMI XCH₄ → агрегация по болотам → enhancement карта
Этап 2: Сопоставление с T_air и NDVI → пространственно-временная динамика
Этап 3: Относительный вклад (болота vs леса)
Этап 4: Абсолютный вклад (валидация по наземным станциям)
Этап 5: Моделирование (опционально)
```

### 3.2. Карта болот первична для атрибуции

Без карты невозможно: отделить болотный сигнал от фона, агрегировать XCH₄ по типам, вычислить относительный вклад.

### 3.3. Наземные данные — для валидации

Потоки CH₄ (Мухрино, Sabrekov Bc8, Бакчар, ZOTTO) проверяют TROPOMI-based оценки. Порядок: сначала TROPOMI-картина, потом сопоставление с наземными.

### 3.4. Только опубликованные данные

Из рецензированных публикаций и открытых датасетов. Неопубликованные — с согласием PI.

### 3.5. Честность относительно ограничений

Column-averaged ≠ surface; зима не покрыта; ~7 км не различает микроландшафты; 1–3 станции не покрывают всю зональность.

### 3.6. Открытость и воспроизводимость

Код открыт, GEE App публичен, данные доступны.

### 3.7. Никаких overclaims

Инструмент исследовательский. Без заявлений о верификации кадастров.

## 4. Критерии качества (аксиология)

### 4.1. Карта болот
- OA ≥ 85%, ≥ 5 типов, покрытие 55–68°N, 60–85°E

### 4.2. TROPOMI агрегация
- Месячные композиты XCH₄ по болотам — без артефактов, ≥ 100 валидных пикселей на месяц
- Сезонная динамика воспроизводится (пик июль–август)
- Разница XCH₄ (болота − леса) > 0 ppb летом (если нет — метод не работает)

### 4.3. Валидация
- Корреляция сезонного хода XCH₄ с наземными потоками ≥ 0.5
- Сопоставимость с Bc8 по порядку величины

### 4.4. Приоритеты

```
Научная корректность > Работающий TROPOMI пайплайн > Карта болот > Валидация > Моделирование > Интерфейс
```

## 5. Принципы (праксеология)

### 5.1. Экономическая модель
Бесплатный. GEE + Copernicus + ERA5 — открытые данные.

### 5.2. Масштаб
- Территория: ЗС (55–68°N, 60–85°E)
- Время: 2019–present
- Разрешение: месячные композиты

### 5.3. Эволюция — инкрементальная

```
1. TROPOMI: собрать, отфильтровать, агрегировать по месяцам
2. Карта болот: маска + типология
3. Интеграция: XCH₄ по болотам vs по лесам → относительный вклад
4. Сопоставление с T_air, NDVI → пространственно-временной паттерн
5. Валидация по наземным данным (Мухрино, Sabrekov, ZOTTO)
6. Оценка абсолютной эмиссии (transfer function)
7. Моделирование (если данных достаточно)
8. GEE App + публикация
```

## 6. Специфика предметной области

### 6.1. Три показателя для сопоставления

| Показатель | Источник | Роль |
|-----------|----------|------|
| XCH₄ / ΔCH₄ | TROPOMI | Уровень метана в атмосфере |
| T_air (2м) | ERA5 | Управляющий фактор метаногенеза |
| NDVI | MODIS/S2 | Proxy продуктивности, субстрат для метаногенов |

Эти три показателя извлекаются для болот и лесов, сопоставляются во времени и пространстве, затем валидируются по наземным станциям.

### 6.2. Относительный vs абсолютный вклад

**Относительный:** ΔCH₄_болота − ΔCH₄_леса. Не требует физической конверсии. Показывает, насколько болота добавляют метана по сравнению с лесным фоном. Самый надёжный результат.

**Абсолютный:** оценка surface flux через сопоставление TROPOMI с наземными данными. Transfer function: flux = f(ΔCH₄, T_air, NDVI). Менее надёжный, но даёт Тг/год.

### 6.3. Оценки-референсы

| Источник | Метод | Оценка |
|----------|-------|--------|
| Glagolev et al. 2011 | Bottom-up Bc8 | 3.21 Тг C-CH₄/год |
| Sabrekov et al. 2013 | Bottom-up камеры | 1.08 Тг C (сев.+средн. тайга) |
| Kim et al. 2011 | Top-down инверсия | ~6 Тг CH₄/год |
| Yuan et al. 2024 | ML upscaling | Пик 2016 — ЗС |

### 6.4. Мухрино — главная точка валидации

60.89°N, 68.68°E. EC CH₄ с 2019, автокамеры КАСМ8. Метео 2010–2019 на Zenodo. Данные CH₄ из Dyukarev et al. 2024.

## 7. Типовые запросы

1. «Каков уровень XCH₄ над болотами ЗС в июле 2023?» → Карта + среднее значение
2. «Насколько болота отличаются от лесов по CH₄?» → ΔCH₄ (болота − леса) по месяцам
3. «Какие типы болот дают наибольший enhancement?» → Boxplot по типам
4. «Совпадает ли сезонный ход XCH₄ с наземными данными?» → Два ряда на одном графике
5. «Какова суммарная эмиссия?» → Оценка через transfer function (этап 6)
