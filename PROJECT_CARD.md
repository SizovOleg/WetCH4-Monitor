## 1. КОНЦЕПТ

### Идея
GEE-инструмент мониторинга эмиссии CH₄ из болот Западной Сибири по данным TROPOMI с привязкой к типологии болот и валидацией по наземным станциям.

### Проблема
Болота ЗС — крупнейший природный источник CH₄ в высоких широтах. Глобальные продукты не имеют калибровки в регионе. Регионального инструмента нет.

### Решение
Три показателя (XCH₄, T_air, NDVI) сопоставляются с картой болот → относительный вклад (болота vs леса) и абсолютный (через наземные станции).

### Критерии успеха
1. TROPOMI месячные композиты работают в GEE
2. Карта болот ≥ 5 типов, OA ≥ 85%
3. Enhancement болота > леса > 0 ppb летом
4. Сезонный ход XCH₄ коррелирует с наземными (r ≥ 0.5)
5. Статья принята в рецензируемый журнал

### Чем НЕ является
- Не оперативная система оповещения
- Не инструмент верификации кадастров
- Не полная атмосферная инверсия

## 3. ФУНКЦИОНАЛЬНОЕ ОПИСАНИЕ

### Модуль 2: tropomi_monthly (ПЕРВЫЙ В РАЗРАБОТКЕ)

**Роль:** Получить чистые месячные карты XCH₄ над ЗС.

**Что делает:**
- Загружает COPERNICUS/S5P/OFFL/L3_CH4
- Фильтрует по AOI, датам, qa_value > 50
- Агрегирует в месячные медианные композиты (май–октябрь)
- Считает n_observations на пиксель

**Выход:** ee.ImageCollection, band 'xch4' (ppb), properties year/month/n_obs

### Модуль 1: wetland_mask

**Роль:** Пространственная типология болот для маски и атрибуции.

**Что делает:**
- Медианный композит Sentinel-2 (июнь–август)
- Спектральные индексы: NDVI, NDWI, MNDWI, NDMI
- RF-классификатор по обучающей выборке
- 9 классов (0=не болото, 1–8=типы Bc8)

### Модуль 3: enhancement

**Роль:** Определить, насколько XCH₄ над болотами выше, чем над лесами.

**Что делает:**
- Для каждого месячного композита: средний XCH₄ по wetland_type > 0 и по wetland_type == 0
- ΔCH₄ = XCH₄_wetland − XCH₄_forest
- Карта enhancement: попиксельно XCH₄ − background (медиана по не-болотным)

**Kill signal:** если ΔCH₄ ≤ 0 для июля → метод не работает.

### Модуль 4: three_variables

**Роль:** Совместный анализ CH₄ + T_air + NDVI.

**Что делает:**
- Добавляет ERA5 T_air (2м) и MODIS NDVI к месячным композитам
- Выгружает таблицу: month, xch4_wetland, xch4_forest, delta_ch4, t_air, ndvi
- Scatter plots: ΔCH₄ vs T_air, ΔCH₄ vs NDVI

### Модуль 5: validation

**Роль:** Проверка по наземным данным.

**Что делает:**
- Извлекает XCH₄ из TROPOMI для координат станций
- Строит сезонный ход: TROPOMI XCH₄ vs наземный flux
- Корреляция, scatter plot

### Модуль 6: emission_estimate

**Роль:** Оценка абсолютной эмиссии.

**Что делает:**
- Transfer function: flux = f(ΔCH₄, T_air, NDVI) по наземным данным
- Применяет к территории
- Интегрирует → Тг CH₄/год
- Сравнивает с Bc8 (3.21 Тг C)

### Модуль 7: app

**Роль:** Интерактивный интерфейс.
